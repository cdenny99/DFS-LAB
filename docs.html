⚾ MLB Workflow

  1. Upload the DraftKings slate CSV.
  2. Run the MLB builder/trimmer notebook.
  3. Review generated lineups and exposure report.
  4. Select the final portfolio based on stack quality and exposure balance.

⚾ MLB Notes

  • Primary focus: stacked DraftKings lineups.
  • Uses Name(ID) export format.
  • Avoids hitters against opposing starting pitchers.
  • Includes exposure reporting.

⚽ Soccer Workflow

  1. Run the Player Ratings Generator.
  2. Use the enriched ratings CSV in the Lineup Builder.
  3. Generate DraftKings lineups.
  4. Run the Lineup Trimmer.
  5. Review final portfolio and exposures.

⚽ Soccer Notes

  • Salary range generally targets $49,000–$50,000.
  • Minimum three teams required.
  • Goalkeeper opponent controls are included.
  • Portfolio trimming and exposure reporting are included.
