⚾ MLB Workflow

Production

  1. Upload the DraftKings Slate CSV.
  2. Run the MLB Builder / Trimmer notebook.
  3. Review generated lineups.
  4. Review exposure report.
  5. Select final portfolio.

⚾ MLB Notes

  • Primary focus is stacked DraftKings lineups.
  • Name(ID) export format.
  • No hitters against opposing starting pitchers.
  • Exposure reporting included.
  • Portfolio trimming included.

⚽ Soccer Workflow

  1. Run Ratings Generator.
  2. Create enriched ratings CSV.
  3. Run Lineup Builder.
  4. Generate DraftKings lineups.
  5. Run Lineup Trimmer.
  6. Review final exposures and portfolio.

⚽ Soccer Notes

  • Typical salary range: $49,000–$50,000.
  • Minimum 3 teams.
  • Goalkeeper opponent controls enabled.
  • Portfolio trimming and exposure reporting included.
  • Supports 20-max and single-entry workflows.
